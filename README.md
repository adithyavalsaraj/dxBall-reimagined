# DX Ball Neo

A modern, high-performance reimagining of the classic DX-Ball arcade game, built with React, TypeScript, and Vite.

## Features

- **Classic Gameplay**: Experience the nostalgic brick-breaking action with modern physics and smooth animations.
- **Dynamic Levels**: Challenging level designs with various brick types and patterns.
- **Responsive Design**: Playable on multiple screen sizes thanks to a robust canvas implementation.
- **Premium Aesthetics**: Sleek dark mode interface with vibrant effects and smooth transitions.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: [Vanilla CSS / Tailwind CSS]
- **Rendering**: HTML5 Canvas API

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/adithyavalsaraj/dxBall-reimagined.git
   cd dx-ball-neo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Gameplay Instructions

- **Move Paddle**: Mouse movement or keyboard arrows.
- **Launch Ball**: Click or Spacebar.
- **Goal**: Clear all bricks on the screen without letting the ball fall below the paddle.
